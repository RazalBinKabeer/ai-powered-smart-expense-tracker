# AI-Powered Smart Expense Tracker

An intelligent application that helps you track, categorize, and analyze your expenses using AI. Built to be fast, responsive, and easy to use.

## Features (Developing)
- **Smart Categorization**: Automatically categorize expenses using AI.
- **Insights & Analytics**: Gain actionable insights into your spending habits with clear, dynamic charts.
- **Responsive Design**: Beautiful, responsive UI that works seamlessly across desktops and mobile devices.

## Tech Stack
- [Next.js](https://nextjs.org/)
- [React](https://react.dev/)
- [Tailwind CSS](https://tailwindcss.com/)
- TypeScript

## Getting Started

First, install the dependencies if you haven't already:

```bash
npm install
```

Then, run the development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the application in action.

You can start editing the page by modifying `app/page.tsx`. The page auto-updates as you edit the file.
